Getting Started

Thank you for purchasing BizInvoicer! This comprehensive invoice management system is built with Node.js, Express, and MySQL. It provides a robust solution for businesses to manage their invoicing, client relationships, and financial documentation.

Important: This is a commercial script. Please ensure you have a valid license key before installation.

Prerequisites

  • Node.js (v14 or higher)
  • MySQL (v8.0 or higher)
  • npm or yarn package manager
  • Valid license key (provided with purchase)

Installation

Follow these steps to get BizInvoicer up and running:

  1. Extract the ZIP file:

    Extract the downloaded ZIP file to your desired location on your server.

  2. Create Database in Your Database:
  3. Update Config file: 
    {
    "development": {
        "username": "DB_USERNAME",
        "password": "DB_PASSWORD",
        "database": "DB_NAME",
        "host": "DB_HOST",
        "dialect": "mysql"
    }
}
  4. Update .env file: 
    NODE_ENV=development
SESSION_SECRET=YOUR_SECRET
BASE_URL=YOUR_WEBSITE_URL
PORT=4000
  5. Init Database: 
    npm run database-init
  6. Install dependencies: 
    npm install
  7. Start the application: 
    npm start

The application will be available at http://your-host:4000

Note: Make sure to keep your license key secure and never share it publicly. The license key is required for the application to function properly.

Technology Stack

  • Backend Framework: Node.js with Express
  • Database: MySQL with Sequelize ORM
  • Template Engine: EJS
  • Authentication: Express Session, bcryptjs
  • API Documentation: Swagger UI
  • File Upload: Multer
  • Email Service: Nodemailer
  • Security: Helmet

Updating BizInvoicer

When a new version of BizInvoicer is available, follow these steps to update your installation:

Important: Always backup your database and files before performing an update.
  1. Download the new version:

    Download the latest version ZIP file from your purchase dashboard.

  2. Backup your current installation:
    • Backup your database
    • Backup your `.env` file
    • Backup any custom modifications you've made
  3. Replace the files:

    Extract the new ZIP file and replace all files in your current installation directory. Make sure to:

    • Preserve your existing `.env` file
    • Keep your custom modifications if any
    • Don't overwrite your database files
  4. Update dependencies: 
    npm install
  5. Run database migrations: 
    npx sequelize-cli db:migrate
  6. Restart the application: 
    npm start
Note: If you encounter any issues during the update process, please contact our support team at foreversolutions23@gmail.com
